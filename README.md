# EMAIL CHECK
𝐄𝐦𝐚𝐢𝐥 𝐂𝐡𝐞𝐜𝐤, using Holehe in Flask to perform OSINT on email addresses. 🕵🏽‍♂️ <br>
Check on which platforms an email is linked to for your OSINT investigations. 👨🏽‍💻
